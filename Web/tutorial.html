TiRank-web Tutorial

Welcome! This guide will walk you through the entire TiRank-web application, from uploading data to analyzing your results.

What is TiRank-web?

The TiRank-web application is a data-driven tool for creating and deploying powerful data analysis applications. It is structured into six key sections: Homepage, Upload Data, Pre-processing, Analysis, Tutorial, and FAQs.

How This Guide is Structured

This tutorial will walk you through the four main stages of the app:

  1. Upload Data: Loading your datasets into the application.
  2. Pre-processing: Cleaning and preparing your data for the model.
  3. Analysis (TiRank): Running the main TiRank model.
  4. Analysis (Downstream): Performing DEG and pathway enrichment.

Let's get started!

Step 1: Uploading Your Data

1.1 Load Example Data (Quickstart)

To help you get started, we provide example datasets for both Spatial Transcriptomics (ST) and Single-cell RNA sequencing (scRNA-seq).

As shown in the image, you can select 'Spatial Transcriptomics Data' or 'scRNA-seq Data' to automatically load our sample data. This will fill in all the required paths for the rest of this tutorial.

Hint: You only need to fill out one of the ST (2-1) or scRNA-seq (2-2) forms, not both.

1.2 Uploading Bulk Transcription Data

We will use the loaded ST data as our example.

  1. Enter Paths: The 'Expression Matrix' and 'Clinical Information' fields will be auto-filled from the example data. To use your own data, you must enter the absolute paths to your files here.
  2. View Data: Click the 'View' button to preview your loaded tables in the 'View Data' card on the right.
  3. Check Data: Click the 'Check' button to validate that your data format is compatible with the TiRank model.

1.3 Uploading ST or scRNA-seq Data

This step specifies which type of data you are using.

  1. Enter Paths: This will also be auto-filled by the example data. To use your own data, enter the absolute paths here.
  2. View Data: As before, click 'View' to preview your data.
  3. Confirm Type: Click the 'Confirm' button (in either the 2-1 card for ST or the 2-2 card for scRNA-seq). This step is essential because our app processes ST and scRNA-seq data differently.

Step 2: Pre-processing Your Data

2.1 Normalization and Transformation

  1. Select Options: In the 'Pre-processing Data' card, choose "Enable" or "Bypass" to decide whether to perform Normalization and Log-transformation.
  2. Perform: Click the 'Perform' button to start pre-processing.

Note: The system will show a "Loading" screen. This step may take a few minutes. Please wait for it to complete before clicking anything else.

2.2 Mode Selection and GenePair Transformation

  1. Mode select: Choose your desired mode.

  2. GenePair Transformation: Select values for 'Top var genes', 'P value threshold', and 'Top gene pairs'. We recommend these default parameters:

    'Top var genes': 2000
'P value threshold': 0.05
'Top gene pairs': 2000

    Click the 'Perform' button to run the transformation.

Note: This is another long step. The system will enter a "Loading" screen. Please be patient and do not navigate away.

2.3 Viewing Pre-processing Results

After the pre-processing and transformation are complete, you can use the radio buttons in the right-hand card to explore the different output plots.

Step 3: Analysis with TiRank

3.1 Device Selection (CPU vs. GPU)

First, choose the hardware you want to use for training. In the 'Device' dropdown, you can select Cpu or Cuda (for GPU training).

If you want to use a GPU, you must have the correct PyTorch version installed.

  1. Check your GPU driver version in your terminal:
    nvidia-smi
  2. Check if PyTorch can detect your GPU by running this Python code:
    import torch
print(torch.cuda.is_available())

If the output is True, you are ready to use your GPU. If False, you must install a compatible version of PyTorch from the Pytorch official website.

3.2 Training the TiRank Model

  1. Set Parameters: You can click 'advanced' to change the model's hyperparameters. If you don't, the following defaults will be used:
    'Nhead': 2
'n_output': 32
'nhid1': 96
'nhid2': 8
'nlayers': 2
'n_trails': 20
'dropout': 0.5
  2. Train: Click the 'Train' button. This is the longest step, so please be patient.

3.3 Predicting with the TiRank Model

  1. Set Parameters: You can select 'Reject' to perform rejection and 'advanced' to change prediction parameters. The defaults are:
    'Tolerance': 0.05
'Reject_mode': 'GMM'
  2. Predict: Click the 'Predict' button.

3.4 Viewing TiRank Results

When training and prediction are finished, you can select and view the resulting graphs from the drop-down box in the right-hand card.

Step 4: Downstream Analysis (DEG & Pathway)

The final section performs differential gene expression and pathway enrichment.

  1. Differential Expression Genes (DEG): Select your desired logFC threshold and P-value threshold from the dropdowns and click "Perform".
  2. Pathway Enrichment: Simply click the 'Perform' button to run the analysis.
  3. View Results: The results will appear on the right, where you can click the 'download' button to save them.
